RP2.htmについて

RP2.htmはhtmファイルです。
このファイルをShockWave for DirectorプラグインがインストールされているWWWブラウザで開くと、ShockWaveムービーが楽しめます。
以下、標準的なWindows95環境で、インターネットエクスプローラがインストールされているマシンを想定して、ご説明します。

1）デスクトップ上の「マイコンピュータ」アイコンをダブルクリックして開きます。
2)「ROLY_2」アイコンを右クリックして「開く」を選びます。
3)「HTML」フォルダ内の「RP2.htm」をダブルクリックします。
4)マシンに設定されているWWWブラウザが起動し、つづいて、RP2.htmの内容が表示されます。

このとき、ブラウザの設定によっては「安全性不明なデータがあるかActiveXコントロールが含まれていますが・・」というアラートが表示されることがありますが、これは、ショックウエーブプラグインを使用しているためですので、「すべて実行する」をクリックしてください。

もしも、この作業がうまくいかなかった場合は、はじめに、ブラウザを起動しておき、ファイルメニューの「開く」から「RP2.htm」を開くことで実行できます。
あなたのマシンにShockWaveプラグインがインストールされていない場合は、ファイルをひらいても、アニメーションが表示されません。その場合は、お手数ですが
